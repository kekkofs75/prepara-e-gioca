# Le Carte

Ogni carta ha **due numeri diversi**, uno per ciascun orientamento (lato top/bottom). Il numero visibile in alto a sinistra quando tieni le carte in mano è il valore "attivo" di quella carta per quel round.

Girando fisicamente tutta la mano all'inizio del round, i numeri sull'altro orientamento diventano quelli attivi. **Durante il round non puoi riordinare le carte né cambiarne l'orientamento individualmente.**

# Tipi di Set

## Numeri consecutivi
Le carte del set mostrano numeri in sequenza, senza salti. L'ordine crescente o decrescente è indifferente.

*Esempi validi: 4-5-6 / 9-8-7-6 / 2-3*
*Esempio non valido: 4-6-7 (manca il 5)*

## Numeri abbinati
Tutte le carte del set mostrano lo stesso numero.

*Esempi validi: 3-3 / 7-7-7*

# Cos'è un Set più Forte

Quando mostri un set, deve superare il **Set Attivo** corrente. Il confronto segue tre criteri in ordine di priorità:

**1. Quantità** — Il set con più carte è il più forte.
Se il tuo set ha più carte del Set Attivo puoi mostrarlo; se ne ha di meno, non puoi.

**2. Tipo** *(a parità di quantità)* — I numeri abbinati battono i numeri consecutivi.
Se entrambi i set sono dello stesso tipo, passa al criterio successivo.

**3. Numero minimo** *(a parità di quantità e tipo)* — Confronta il numero più basso di ciascun set. Se il tuo è **strettamente più alto**, puoi mostrare. Se è uguale o più basso, non puoi.

> **Attenzione al pareggio**: se i numeri minimi sono uguali, il set sfidante *non* supera il Set Attivo e la Mostra non è consentita.

# Il Set Attivo

È il set di carte visibile sul tavolo, giocato per ultimo tramite una Mostra. Ha sempre un **proprietario** — il giocatore che lo ha mostrato.

- Quando qualcuno Esplora da esso, il proprietario riceve 1 Segnalino Scout.
- Le carte Esplorate vengono inserite nella mano dell'esploratore (non collezionate come punteggio).
- Se il Set Attivo si svuota, il prossimo giocatore può mostrare qualsiasi set.
